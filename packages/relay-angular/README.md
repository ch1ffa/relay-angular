# relay-angular
Relay for Angular 

## Installation

Install relay-angular using yarn or npm:

```
yarn add relay-angular relay-runtime
```

Install relay-compiler using yarn or npm:

```
yarn add relay-angular-plugin ngx-build-plus relay-compiler relay-config
```
